##############################Описание################################## 
 В качестве среды разработки использовалась Microsoft Visual Studio 2015.
 В проекте используется многоуровневая архитектура:
	- Automate.DAL - Data Access Layer;
	- Automate.BLL - Business Logic Layer;
	- Automate.WEB - Presentation Layer.
 Используется механизм внедрения зависимостей (IoC-контейнер Ninject).
 Файл базы данных "Automate.mdf" располагается в каталоге "\Automate.WEB\App_Data".
 Файл "\Automate.WEB\Files\nophoto.png" используется при первичной инициализации БД.
 Для доступа в панель администратора используется параметр "key" со значением "automate".
 
 В проекте выполнена функция импорта напитков. Данный функционал реализован путем получения данных из "Excel" файла 
 (пример файла прилагается: "\Import.xlsx").
 Также реализован функционал выбора нескольких напитков перед получением сдачи.
